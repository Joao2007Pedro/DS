# ☕ Exercícios de POO em Java

Repositório com exercícios da disciplina **POO (Programação Orientada a Objetos)** do **2º ano do curso de Desenvolvimento de Sistemas (DS) da ETEC**.

Todos os exemplos e exercícios foram desenvolvidos **100% em Java**, com foco na prática dos principais conceitos de **Programação Orientada a Objetos**.

---


## ☕ Linguagem Utilizada

* **Java**

---

## 📚 Sobre o Repositório

Durante as aulas, os exercícios e exemplos foram sendo organizados em pastas conforme os conteúdos eram apresentados.
Por isso, a estrutura do repositório segue principalmente o formato de **aulas e exemplos práticos**, e não necessariamente uma organização rígida por tema.

Isso permite acompanhar a **evolução dos conceitos aprendidos ao longo das aulas**.

---

## 📂 Estrutura do Repositório

O repositório contém pastas como:

* `Classe Abstrata`
* `Encapsulamento`
* `Get e Set`
* `Herança`
* `Polimorfismo`
* `Interface`
* `Sobrecarga`
* entre outras relacionadas aos conteúdos estudados.

Cada pasta contém **exemplos, exercícios ou pequenos projetos feitos durante as aulas**.
